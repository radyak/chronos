Administration: Tags

Tags

Entries can be attributed with tags, to allow for grouping, filtering or adding additional, structured data to entries. Example: The entry for a ruler may have the tags for Emperor and Holy Roman Empire assigned.

Tag Categories

Tags need a category to categorize them. Example: a tag 'Roman' could refer to the culture, the territory, a period etc. - the corresponding category helps to establish the correct context.
